## 1. Seasons of Pelican Hearts
**[Open](https://cctail.github.io/codex-portfolio/Seasons%20of%20Pelican%20Hearts.html)**

A fully playable browser visual novel — zero dependencies, zero assets. All pixel art is generated procedurally via Canvas API (buildings, hills, characters, seasonal particles); all audio via Web Audio API (chord voicing, envelope shaping, per-heroine music themes). Three complete character routes, affection meters, branching endings, save/load, dialogue backlog, auto/skip modes.

Built through ~20 rounds of iterative Codex-assisted development. I directed Codex like a junior engineer: write the spec, review the output, catch the regressions, iterate until it ships.

## 2. BaZi Fortune Teller
**[Open](https://cctail.github.io/codex-portfolio/BaZi%20Fortune%20Teller.html)**

A domain-expert system encoding 1,500 years of Chinese metaphysical logic in client-side JS — no backend, no dependencies, single HTML file. 2,000+ lines of pure vanilla JS implementing a real astronomical algorithm (S_TERM_INFO) for solar term calculation across 200 years, a full 60-cycle Jiazi calendar, weighted hidden stem scoring, and polarity-aware Ten God logic from scratch. Seven analysis modules, detection of 合冲刑 interactions and five 神煞 types, full Chinese/English i18n toggle, and a built-in validation panel running known-correct test cases on page load. Every output computed from first principles — no oracle, no lookup table.

## 3. Developer Productivity Index
**[Open](https://cctail.github.io/codex-portfolio/developer-productivity-index.html)**

An interactive dashboard defining and instrumenting developer productivity for an AI coding tool. Covers North Star metrics (session productivity, coding capacity utilization, net time saved), guardrail metrics (latency, error rate, regression), and drill-downs by language, repository size, and task type. Mirrors the measurement framework I built for Meta's internal AI coding assistant.
